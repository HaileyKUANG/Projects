Part3. About rmarkdown
If you are using RStudio, you can install the R Markdown package from CRAN as follows:
install.packages("rmarkdown")

If you are not using RStudio, please use the following command:
rmarkdown::render("C:/Users/.../R_Data_Analysis_Fraud_Transaction_with_Regression.Rmd")
Please replace the path with /.../

You can also find the R Markdown Quick Tour at:
https://rmarkdown.rstudio.com/authoring_quick_tour.html
